# AI Environment

A reproducible Python environment for AI, Deep Learning, and Seismology.

## Requirements

- Linux
- Miniforge or Miniconda
- NVIDIA driver 595+
- NVIDIA GPU (optional)

## Installation

```bash
./install.sh
```

## Activate

```bash
conda activate ai
```

## Verify

```bash
python verify.py
```

## Update

```bash
mamba env update -f environment.yml

conda activate ai

pip install -r requirements.txt
```
